# Buenos Días Café x Por La Noche

A restaurant website concept built for a real small business opening in Grand Prairie, TX.

The site is designed around a dual experience — serving coffee and breakfast during the day, then reopening in the evening for dinner and a more elevated atmosphere.

## Live Site
https://codingwithandres.github.io/buenos-dias-cafe-site/

## Features
- Multi-page layout (Home, Day Menu, Night Menu, About, Contact)
- Responsive design
- Real business location and contact section
- Embedded Google Map

## Tech
HTML, CSS, GitHub Pages

## Status
In progress. Currently refining layout, content, and overall design.
